关于小莺

ABOUT US

小莺出国隶属于远英出国服务有限公司。注册资本5000万人民币。总部位于中国上海，
在东京、首尔、新加坡、南京、苏州、成都、石家庄，郑州、香港等地设有分公司。多年来，小莺出国留学服务团队秉承专业贴心
的服务理念，利用大数据互联网技术大幅加速了留学行业的效率化，智能化、云端化。基于深厚的互联网基因，小莺并没有像其他互联网留学
机构一样止步于网络申请、为提高服务质量，小莺在全国开设有二十余家体验中心，由总部对体验中心咨询人员进行一流的培训和管理，提高本地服务
质量。目前为止，小莺累计服务学院近50万名，小莺网校与小莺出国平台注册用户超过500万人。

以用户为导向的新型留学机构

NEW STUDY INSTITUTION

传统留学机构为便于自身管理，往往以服务流程为导向，简而言之，就是忽略用户的需求与便利性，最大限度的追求自身业务
流程的无缝对接，形成了专业的“流水线作业体系”，大幅降低了留学客户体验和个性化服务品质。通过大规模对背景普通的咨询人员进行机械化的培训快速
上岗，形成了走系统，走流程的行业通病，难以满足日益提高的个性化留学服务需求。

专业   贴心   新留学

贴心"是小莺出国处理客户关系时的重要标准。在小莺，每一位学员和申请者都是需要精心培养和照料的"小宝宝"，从申请和培训的每一个环节，细致入微的为他们提供贴心的帮助十分重要，留学是一项相对复杂的系统工程，我们的顾问和老师为大家保驾护航。

SLAS智能云申请与教学系统

SLAS INTELLIGENT CLOUD APPLICATION AND TEACHING SYSTEM

基于现金的IT技术，小莺开发团队创造性的开发了一整套完整的SLAS智能云申请与教学
服务系统Smarty Learning&Applying System。该系统基于用户行为分析，大幅优化了留学申请与培训流程，使整个申请过程智能化，
云端化，效率化。同时为每一位留学咨询人员提供了强有力的申请数据支持，系统囊括了数以万计的海外院校最新招生数据，申请教授数据，录取案例数据
等，帮助咨询人员作出最专业的留学规划，同时保证申请流程畅通无阻，高效互联。

发展历程

DEVELOPMENT PATH

公司发展大事记：2010年小莺出国正式在上海开设了总部，成立远英出国有限公司，2012年香港万道集团注资1500万，
2014年开设网络申请教学系统1.0版正式上线，2016年开设了日本，韩国，香港，新加坡等国家申请服务，2017年服务学生数量业内领先，2018年开设的
分校已经达到20多家分校，在今后的小莺出国会越来越好。
